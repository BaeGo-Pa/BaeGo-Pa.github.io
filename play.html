Instructions d'installation
Windows

Avec l'installeur

  1. Télécharger l'installeur
  2. Double-cliquer sur l'installeur
  3. Choisir si le jeu doit-être installé seulement sur la session active (recommandé), ou pour tous les utilisateurs (nécessite les droits administratifs)
  4. Choisir la langue de l'installeur
  5. Suivre les instructions de l'installeur
  6. Et vous pouvez y jouer !
Désinstaller le jeu
Windows

Depuis le menu Démarrer

  1. Chercher le dossier Jiwa Peteng
  2. Faire clic droit sur le logiciel puis Désinstaller
  3. Chercher Jiwa Peteng version n dans la liste des programmes, avec n la version du jeu
  4. Faire clic droit sur le programme puis "Désinstaller", ou clic gauche puis desinstaller au-dessus de la liste des programmes

Depuis le Panneau de configuration

  1. Sous Programmes, cliquer sur Désinstaller un programme
  2. Chercher Jiwa Peteng version n dans la liste des programmes, avec n la version du jeu
  3. Faire clic droit sur le programme puis Désinstaller, ou clic gauche puis desinstaller au-dessus de la liste des programmes

Depuis le dossier de Jiwa Peteng

  1. Double-cliquer sur le fichier unins000.exe
